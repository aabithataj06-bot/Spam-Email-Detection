Spam Email Detection 📧
This project detects whether a message is **Spam or Not Spam** using Machine Learning.
🔧 Technologies Used
* Python
* Pandas
* Scikit-learn
* NLP (CountVectorizer)
* Naive Bayes
*  📊 Accuracy
Achieved around 97% accuracy
 🚀 Features
* Classifies messages as Spam / Not Spam
* Uses NLP techniques
* Simple and efficient model
 📁 Dataset
SMS Spam Collection Dataset
▶️ How to Run
1. Open Jupyter Notebook
2. Run all cells
3. Test your own messages
